# Project: FPL Manager Stats Telegram Bot

# Description:
# Telegram bot displaying Fantasy Premier League manager statistics.
# Send a manager ID to receive points, rankings, team lineups,
# league standings, and historical performance.

# Features:
# - Current gameweek points and overall rankings
# - Team lineup with individual player points and captain points
# - League standings with manager's rank
# - Gameweek navigation with interactive buttons (1-38 circular)
# - Simple view and detailed view modes
# - Historical performance from previous seasons

# Requirements:
# - Python 3.11+
# - Libraries: python-telegram-bot==20.7, requests
# - Hosting: Railway.app or any Python cloud platform
# - Telegram Bot Token from @BotFather

# Installation:
# 1. git clone <repository>
# 2. pip install -r requirements.txt
# 3. Set environment variable: BOT_TOKEN=your_token
# 4. python main.py

# Deploy on Railway.app:
# 1. Push code to GitHub
# 2. Connect Railway to GitHub repository
# 3. Add BOT_TOKEN as environment variable
# 4. Set start command: python main.py
# 5. Deploy

# Get Manager ID:
# 1. Visit https://fantasy.premierleague.com
# 2. Go to any manager's points page
# 3. Copy ID from URL: https://fantasy.premierleague.com/entry/1234567/
# 4. Send this number to the bot

# Bot Commands:
# - Send any manager ID number
# - /start or /help - Show help message

# Interactive Buttons:
# - 📋 Simple View: Basic statistics
# - 📊 Detailed View: Complete statistics
# - ⬅️ Previous Gameweek: Navigate back
# - ➡️ Next Gameweek: Navigate forward (circular)

# Data Source:
# Official FPL API: https://fantasy.premierleague.com/api/

# License: Open source for personal and educational use.
